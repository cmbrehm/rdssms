Creating a lab environment using AWS CloudFormation

To simplify the getting started experience with the labs, we have created foundational templates for AWS CloudFormation that provision the resources needed for the lab environment. These templates are designed to deploy a consistent networking infrastructure, and client-side experience of software packages and components used in the lab.

Formal Event
If you are running these labs in a formal instructional event, the lab environment may have been initialized on your behalf. If unsure, please verify with the event support staff.

Please use the template to initialize you environment by clicking on the Launch Stack button below. If you would like to look at the template, you can also download it.

Launch Template Download Template
cfr-rdsSQL.yml

Region Check
Ensure you are still working in the correct region, especially if you are following the link above to open the service console at the right screen.

From the Create stack page create the Stack by uploading the cloudformation file.

In the Specify stack details provide information

Stack name: sqlimmersion-stack

WinKeyName: sqlimmersion (This is the keypair that you created in previous lab)

Scroll to the bottom of the page and click Next.

Scroll to the bottom of Configure stack options page and click on Next.

On the Review sqlimmersion-stack page, scroll to the bottom of the page and then click Create stack.

The stack will take approximatively 5 minutes to provision, you can monitor the status on the Stack detail page.
